# Miya Software Kurumsal Web Sitesi

Bu proje, **Miya Software** şirketinin dijital dünyadaki yüzü olmak üzere geliştirilmiş modern ve tamamlanmış bir kurumsal web sitesidir. 

## 🚀 Proje Hakkında
Şirketin profesyonel kimliğini, sunduğu hizmetleri ve vizyonunu en iyi şekilde yansıtmayı hedefleyen bu proje; temiz kod yazımı, modern UI/UX prensipleri ve kullanıcı dostu bir arayüz ile tasarlanmıştır.

## ✨ Öne Çıkan Özellikler
- **Çoklu Dil Altyapısı:** `en/` dizini üzerinden sağlanan yapıyla uluslararası kullanıcılara hitap etme imkanı.
- **Modüler Mimari:** Temiz bir klasörleme yapısıyla alt sayfaların (`pages/`), stillerin ve scriptlerin birbirinden ayrıldığı düzenli sistem.
- **Responsive (Duyarlı) Tasarım:** Mobil cihazlardan masaüstü ekranlara kadar her çözünürlükte kusursuz görünüm.
- **Hızlı ve Akıcı:** Ekstra kütüphane bağımlılığı minimumda tutularak saf (vanilla) web teknolojileriyle elde edilen yüksek performans.

## 🛠️ Kullanılan Teknolojiler
Proje, temel web standartları kullanılarak geliştirilmiştir:
- **HTML5** 
- **CSS3** 
- **JavaScript** 

## 📁 Proje Dosya Yapısı
Depodaki dizin mimarisi şu şekildedir:

```text
├── .vscode/          # Çalışma ortamı yapılandırmaları
├── css/              # Sayfa tasarımları ve stil dosyaları
├── en/               # Sitenin İngilizce versiyonuna ait dosyalar
├── images/           # Sitede kullanılan logolar, ikonlar ve fotoğraflar
├── js/               # Animasyonlar ve dinamik işlevler için JavaScript kodları
├── pages/            # Alt sayfalar (Hizmetlerimiz, Hakkımızda, İletişim vb.)
└── index.html        # Sitenin Türkçe ana sayfası
